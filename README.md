# Sistema de Citações

Este projeto foi desenvolvido com o objetivo de praticar HTML, CSS e JavaScript, aplicando conceitos fundamentais do desenvolvimento web na construção de uma aplicação simples e interativa.

O sistema exibe citações em cartões e permite ao usuário personalizar a interface por meio de configurações de acessibilidade, além de criar suas próprias citações.

## Funcionalidades

* Exibição de citações em cartões.
* Adição de novas citações pelo usuário.
* Exclusão de citações criadas pelo usuário.
* Salvamento das citações no Local Storage.
* Restauração da página ao seu estado inicial.
* Modo noturno.
* Modo de alto contraste.
* Ajuste do tamanho da fonte.
* Salvamento das preferências do usuário no Local Storage.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## Conceitos aplicados

Durante o desenvolvimento do projeto foram utilizados conceitos como:

* Estruturação semântica com HTML.
* Organização do layout utilizando Flexbox.
* Manipulação do DOM.
* Criação dinâmica de elementos com JavaScript.
* Tratamento de eventos com `addEventListener()`.
* Manipulação de classes por meio de `classList`.
* Persistência de dados utilizando `localStorage`.
* Estilização de componentes personalizados com CSS.

## Objetivo

O principal objetivo deste projeto foi praticar a manipulação do DOM e o desenvolvimento de funcionalidades comuns em aplicações web, como personalização da interface, criação dinâmica de elementos e armazenamento local de dados.

Além disso, o projeto serviu como exercício para compreender melhor a interação entre HTML, CSS e JavaScript, procurando escrever um código organizado, comentado e de fácil manutenção.

## Possíveis melhorias

* Permitir a edição das citações criadas pelo usuário.
* Adicionar um sistema de pesquisa.
* Organizar as citações por categorias.
* Integrar a aplicação a um banco de dados no futuro.
